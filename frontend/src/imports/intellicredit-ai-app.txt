

Design and generate a **complete production-ready frontend web application** for **Intelli-Credit AI**, an enterprise B2B fintech platform used by credit risk analysts to evaluate companies using AI.

The platform allows analysts to upload financial PDFs, run AI credit analysis, review results, and download a **Credit Appraisal Memo (CAM)**.

Generate **full working code**, not only UI.

---

# Technology Stack

Use the following stack:

```
Next.js 14 (App Router)
TypeScript
TailwindCSS
ShadCN UI
Recharts
Axios
React Hook Form
Lucide Icons
```

Project must be **fully responsive and production ready**.

# Backend API

The application connects to a FastAPI backend with these endpoints:

```
POST /upload
GET /history
GET /evaluation/{id}
```

### POST /upload response

Example response:

```json
{
  "evaluation_id": 12,
  "filename": "company.pdf",
  "financial_data": {
    "revenue": 212143,
    "net_profit": 40121,
    "total_debt": 1504242
  },
  "ratios": {
    "profit_margin": 0.18,
    "debt_to_revenue": 7.09
  },
  "credit_decision": {
    "decision": "APPROVE",
    "interest_rate_percent": 10,
    "suggested_loan_limit": 180
  },
  "explanation": {
    "summary": "Company leverage appears manageable."
  },
  "pdf_path": "outputs/company_CAM.pdf"
}
```

---

### GET /history response

Returns:

```json
[
  {
    "id": 1,
    "company_name": "companyA.pdf",
    "revenue": 300000,
    "risk_score": 72,
    "decision": "APPROVE"
  }
]
```

---

### GET /evaluation/{id}

Returns full evaluation object.
